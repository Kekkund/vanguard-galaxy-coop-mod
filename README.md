# VG Coop — кооп-мод для Vanguard Galaxy

Мод на **BepInEx** для совместной игры вдвоём через Steam. При подключении гость
попадает в мир хоста (общая галактика), и вы видите корабли друг друга.

> Это репозиторий **только с готовым модом**. Исходный код — в отдельном репозитории.

## Скачать

- [**VG_Coop.dll**](VG_Coop.dll) (кнопка **Download**) или [последний релиз](../../releases/latest).

## Установка (без командной строки)

1. Установи **BepInEx** (5.4.x, x64) в игру, если ещё нет. Папка `BepInEx` должна
   появиться рядом с `VanguardGalaxy.exe` (запусти игру один раз, чтобы создались подпапки).
2. Скопируй **`VG_Coop.dll`** в:
   ```
   <папка игры>\BepInEx\plugins\VG_Coop.dll
   ```
3. Запусти игру **через Steam**.

Удаление — просто удали `VG_Coop.dll` из `BepInEx\plugins`.

## Как играть вдвоём — почти всё автоматически

- **Оба** ставят одинаковый `VG_Coop.dll` (или дают моду обновиться самому — см. ниже), запускают игру через Steam, заходят в игру (в космос).
- **Хост:** нажимает **F7** → откроется оверлей Steam → приглашает друга.
- **Друг:** принимает приглашение (или «Join Game» в друзьях Steam).
- **При ПЕРВОМ подключении к этому миру** у друга появится настоящий экран **«Новая игра»**
  (имя, предыстория, выбор корабля — 3D-карусель, как при обычном старте новой партии).
  Он выбирает всё как обычно — и после этого его автоматически внесёт в мир хоста на СВОЁМ
  корабле. Выбор запоминается **именно для этого мира** — при следующем подключении к тому же
  хосту экран создания больше не появится, сразу загрузится этот же кооп-персонаж.
- Если у друга разных миров/хостов несколько — для каждого свой отдельный персонаж.

Единственная обязательная кнопка — **F7** у хоста (начать кооп). Остальное автоматически.

## Автообновление

Мод сам проверяет версию при запуске игры (через `version.txt` в этом репозитории, с проверкой
SHA-256 файла на подлинность). Если вышла новая версия — скачает её, подменит файл и попросит
**перезапустить игру**. Ставить `.dll` руками каждый раз не нужно — только один раз, дальше сам.

## Клавиши

| Клавиша | Действие |
|---------|----------|
| **F7** | Хост: начать кооп + пригласить друга (Steam) |
| **F1** | Открыть/закрыть меню быстрой смены стартового корабля (после первого входа) |
| **F5** | Пинг-маркер «смотри сюда» под курсором |
| **H** | Помахать другу — над твоим кораблём у него всплывёт «Привет! o/» |
| F8 | Отключиться |
| F9 | Диагностика в лог (для отладки) |

**F1 — смена стартового корабля.** После первого входа (когда персонаж уже создан) F1 открывает
короткое меню, где можно поменять стартовый корабль без повторного прохождения «Новой игры».
Первый же выбор персонажа всегда идёт через настоящий экран создания — F1 тут не нужен.

Прочие F-клавиши — служебные/отладочные, игроку не нужны.

## Если что-то пошло не так

Мод пишет собственный диагностический лог рядом с игрой — файл `coop_diag.log`
(и `coop_diag_prev.log`, если игра перезапускалась). Он переживает даже полное зависание
(пишется построчно сразу на диск). Если что-то зависло или повело себя странно — приложите
этот файл при сообщении о проблеме, там будет видно последнее, что успело произойти.

## Требования

- Vanguard Galaxy (Steam), установленный **BepInEx**.
- Оба игрока — друзья в Steam, запуск через Steam, одинаковая версия мода (сверяется автоматически при подключении).

## Что общего у вас двоих

- Общая галактика (гость попадает в мир хоста), видно корабли друг друга; на карте — **у какой станции/поля** друг; в статусе — расстояние до друга.
- Общий кошелёк (кредиты), общие склады станции (арсенал **и** материалы).
- **Миссии по-настоящему общие:** взял один — появилась у обоих; прогресс общий (убийства идут
  через хоста, добыча/сбор сливаются «лучший результат выигрывает»); **сдал любой — засчитано обоим**
  (двойной награды нет). Точки миссий (данжи, лагеря) — одни и те же у обоих.
- **Совместный варп:** уходишь в варп из общей точки — твой корабль ЖДЁТ, а у друга появляется
  кнопка «варпнуться вместе» (10 секунд). Нажал — стартуете одновременно и летите строем.
  Не нажал/не может — тебе сразу придёт отказ, полетишь один. Передумал ждать — кнопка «Лететь одному».
- **Общая добыча:** астероиды скалываются и истощаются одинаково у обоих (лазер/пушки/дрель/ядро),
  контейнеры сальважа — общий прогресс, вскрытые ящики гаснут у обоих как в игре. Руда без дюпа —
  добытое видно обоим, кто подобрал — того и лут.
- Совместный бой: видно ВСЁ оружие друг друга (снаряды, ракеты, все виды лазеров, майнерский луч),
  **общий HP одного врага** (урон складывается), позиции врагов подтягиваются к хостовым, убийства и лут общие.
- Живые события мира (маяки случайных встреч в пути, обломки, боссы по сигнальной ловушке) — общие.
- Приятности: тост «друг прибыл на станцию», предупреждение «друг под огнём!», приветствие по **H**.
- Проверка версий при подключении + автообновление: разные версии почти исключены.

Личное у каждого: инвентарь своего корабля.

## Статус

Весь заявленный костяк готов: общий мир, экономика, склады, лут, добыча, совместный бой,
общие миссии со сдачей на двоих, синхронный варп строем, полноценное создание персонажа гостя
(настоящий экран «Новая игра», привязан к конкретному хосту), автосохранение персонажа гостя
(периодически и при выходе). Автообновление и диагностический лог помогают быстро находить
и чинить проблемы. Идёт финальная обкатка.
