# VG Coop — кооп-мод для Vanguard Galaxy

Мод на **BepInEx** для совместной игры вдвоём через Steam: видно корабль друга
(призрак с именем и HP), в какой он системе, пинг; враги хоста показываются
тенями; есть пинг-маркеры «смотри сюда».

> Это репозиторий **только с готовым модом**. Исходный код — в отдельном репозитории.

## Скачать

- [**Последний релиз**](../../releases/latest) — файлы `VG_Coop.dll` и `VG_Coop_mod.zip`.
- Либо прямо из репозитория: [`VG_Coop.dll`](VG_Coop.dll) (кнопка **Download**).

## Установка (без командной строки)

1. Установи **BepInEx** в игру (если ещё нет). Папка `BepInEx` должна появиться
   рядом с `VanguardGalaxy.exe`.
2. Скопируй **`VG_Coop.dll`** в:
   ```
   <папка игры>\BepInEx\plugins\VG_Coop.dll
   ```
   Обычно: `...\steamapps\common\Vanguard Galaxy\BepInEx\plugins\`
3. Запусти игру **через Steam**.

Удаление — просто удали `VG_Coop.dll` из `BepInEx\plugins`.

## Как играть вдвоём

- **Оба** игрока ставят одинаковый `VG_Coop.dll` и запускают игру через Steam.
- **Хост:** в космосе нажми **F7** — откроется оверлей Steam, пригласи друга.
- **Друг:** прими приглашение (или «Join Game» в друзьях Steam).
- Готово — увидите призраки друг друга, когда будете в одной системе.

## Клавиши

| Клавиша | Действие |
|---------|----------|
| F7 | Захостить + пригласить друга (Steam) |
| F8 | Отключиться |
| F9 | Диагностика (в лог) |
| F6 | Тест соло (без друга): показать призрак-симулятор |
| F5 | Пинг-маркер «смотри сюда» под курсором |

## Требования

- Vanguard Galaxy (Steam), установленный **BepInEx**.
- Оба игрока — друзья в Steam, запуск через Steam.
