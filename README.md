# VG Coop — кооп-мод для Vanguard Galaxy

Мод на **BepInEx** для совместной игры вдвоём через Steam. При подключении гость
попадает в мир хоста (общая галактика), и вы видите корабли друг друга.

> Это репозиторий **только с готовым модом**. Исходный код — в отдельном репозитории.

## Скачать

- [**VG_Coop.dll**](VG_Coop.dll) (кнопка **Download**) или [последний релиз](../../releases/latest).

## Установка (без командной строки)

1. Установи **BepInEx** (5.4.x, x64) в игру, если ещё нет. Папка `BepInEx` должна
   появиться рядом с `VanguardGalaxy.exe` (запусти игру один раз, чтобы создались подпапки).
2. Скопируй **`VG_Coop.dll`** в:
   ```
   <папка игры>\BepInEx\plugins\VG_Coop.dll
   ```
3. Запусти игру **через Steam**.

Удаление — просто удали `VG_Coop.dll` из `BepInEx\plugins`.

## Как играть вдвоём — почти всё автоматически

- **Оба** ставят одинаковый `VG_Coop.dll`, запускают игру через Steam, заходят в игру (в космос).
- **Хост:** нажимает **F7** → откроется оверлей Steam → приглашает друга.
- **Друг:** принимает приглашение (или «Join Game» в друзьях Steam).
- Дальше **само**: мир хоста передаётся другу, друг автоматически загружается в него.
  Вы оказываетесь в одной галактике и видите корабли друг друга.

Единственная обязательная кнопка — **F7** у хоста (начать кооп). Остальное автоматически.

## Клавиши

| Клавиша | Действие |
|---------|----------|
| **F7** | Хост: начать кооп + пригласить друга (Steam) |
| **F5** | Пинг-маркер «смотри сюда» под курсором |
| **F1** | Гость: свой персонаж вкл/выкл (свой корабль вместо копии хоста; экспериментально) |
| F8 | Отключиться |
| F9 | Диагностика в лог (для отладки) |

**F1 — свой персонаж (гость):** по умолчанию гость входит в мир хоста как копия его корабля.
Нажми **F1 в своей игре ДО подключения** — тогда в кооп-мире у тебя будет свой корабль и трюм
(мир, карта и общий склад — хоста), и это сохранится: при следующем подключении загрузится твой
кооп-персонаж. Выключить — снова F1 (вернётся режим копии). Функция новая — если что-то не так,
просто выключи её.

Прочие F-клавиши — служебные/отладочные, игроку не нужны.

## Требования

- Vanguard Galaxy (Steam), установленный **BepInEx**.
- Оба игрока — друзья в Steam, запуск через Steam, одинаковая версия мода.

## Что общего у вас двоих

- Общая галактика (гость попадает в мир хоста), видно корабли друг друга; на карте — **у какой станции/поля** друг.
- Общий кошелёк (кредиты), общие склады станции (арсенал **и** материалы), общие миссии.
- Общий мировой лут: ящик/астероид, вскрытый одним, исчезает и у второго (без дюпа).
- Общий космический лут: выпавший/выброшенный предмет виден обоим, кто первым подобрал — того и лут.
- Совместный бой: видно выстрелы друг друга, убитый одним враг гибнет у обоих.

Личное у каждого: инвентарь своего корабля.
